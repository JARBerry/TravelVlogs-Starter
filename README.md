# TravelVlogs-Starter

## Overview

Video Streaming using AVKit and AVFoundation frameworks.

## Features in this demo

• Use of HLS(HTTP Live Streaming)

• Pause functionality

• Play functionality

• Rewind functionality

• Forward functionality

• Video play back in local mode .mov files
